변수

  1. 변수선언 규칙
     변수명 문자,밑줄,달러기호로 시작
     자바스크립트 예약어 변수화 금지
     대소문자 구분함
  2. let:변수 재할당 가능
  3. const:변수 재할당 불가
  4. var:미친놈임,hoisting함

자료형

  1. number,string,boolean,null,undefined,object,array
  2. typeof():자료형 반환
  3. number+string가능!
  4. 템플릿 리터럴:f string같은거임
     `오늘 사과값은 ${price}원 입니다`

  6. undefined:변수값 선언되었지만 할당되진 않음

객체

  1. object:여러정보 하나에 묶어서 저장 - 딕셔너리임
     let apple{
      color:"red",
      weight:150,
      price:2000
     }
  2. 객체 값 불러오기
     console.log(apple.color)
     console.log(apple["weight"])
  3. 객체 값 추가
     apple.origin = "충주"; //값 추가
  4. 객체 값 수정
     apple.price = 2800; //값 수정
  5. 객체 값 삭제
     delete apple.price;
  6. 배열 선언
     let products = ['고등어','삼치','김치']
  7. 배열 추가
     product.push("딸기")
  8. 배열 맨 뒤 삭제
     product.pop()
  9. 배열 맨 앞 추가
     product.unshift("고구마")
  10. 배열 맨 앞 삭제
     product.shift()

비교 연산자

  1. == : 값이 같으면 참
     5 == '5' -> true
  2. === : 값과 자료형이 같으면 참 ***
     5 === '5' -> false
  3. != : 값이 다르면 참
     5 != '5' -> false
  4. !== : 값이나 자료형이 다르면 참 ***
     5 !== '5' -> true

논리 연산자

  1. && : and
  2. || : or
  3. ! : not

삼항 연산자

  1. let period = (조건) ? 'true 값':'false 값';

요소 접근

  1. document.getElementById:아이디로 document.getElementByClassName: 요소의 클래스명(배열)
  2. document.getElementByTagName: 요소의 태그명(시멘틱 태그)
  3. document.querySelector("선택자") - 첫번째 요소 반환
  4. document.querySelectorAll("선택자") - 모든 요소 반환

콘텐츠 조작

  1. element.textContent: 텍스트 조회,수정
     const winter = document.getElementsByClassName("winter");
     console.log(winter[0].textContent);
  2. element.innerHTML: 텍스트 뿐 아니라 Html 콘텐츠 수정,조회 가능
     const shirt=document.getElementsByTagName("li");
     shirt[0].innerHTML = "Yeah원"
     console.log(shirt[0].textContent)

HTML요소 속성 변경

  1. getAttribute("속성 이름"):속성 조회
  2. setAttribute("속성 이름","새로운 속성값"):속성 수정
  3. setAttribute("disabled",true):
     disabled:button,input,textarea,select등의 폼 요소들 비활성화
  4. removeAttribute("속성 이름"):속성 삭제

이벤트 속성

  1. type:이벤트의 유형
  2. target:이벤트가 발생한 요소(button)

  4. key:눌린 키의 값
  5. clientX,clientY:뷰포트 기준 XY좌표
  6. pageX, PageY:문서기준으로 마우스의 X,Y좌표
